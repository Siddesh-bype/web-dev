# Web Development Projects

This repository contains a collection of web development projects and exercises covering various topics including:

## Projects Included

### HTML Fundamentals
- **2.1 Heading Element** - Basic HTML heading structure
- **2.2 Paragraph Element** - Working with paragraph elements
- **2.3 Void Elements** - Understanding self-closing HTML elements
- **2.4 Movie Ranking Project** - Project combining multiple HTML elements

### Lists and Links
- **3.0 List Elements** - Ordered and unordered lists
- **3.1 Nesting and Indentation** - Proper HTML structure
- **3.2 Anchor Elements** - Creating links
- **3.3 Image Elements** - Working with images
- **3.4 Birthday Invite Project** - Complete HTML project

### CSS Fundamentals
- **5.1 Adding CSS** - Introduction to CSS
- **5.3 CSS Selectors** - Different ways to select elements
- **5.4 Color Vocab Project** - Working with colors

### CSS Layout
- **6.0 CSS Colors** - Color properties and values
- **6.1 Font Properties** - Typography in CSS
- **6.3 CSS Box Model** - Understanding spacing and sizing
- **6.4 Motivation Meme Project** - CSS positioning project

### Advanced CSS
- **7.0 CSS Cascade** - CSS specificity and inheritance
- **7.1 Combining Selectors** - Complex selector patterns
- **7.2 CSS Positioning** - Absolute, relative, fixed positioning
- **7.3 CSS Flag Project** - Advanced positioning project

### Flexbox and Grid
- **8.0 CSS Display** - Display properties
- **9.0 Display Flex** - Flexbox fundamentals
- **9.1 Flex Direction** - Controlling flex direction
- **10.0 Display Grid** - CSS Grid fundamentals
- **10.1 Grid Sizing** - Grid sizing techniques
- **10.2 Grid Placement** - Positioning items in grid
- **10.3 Mondrian Project** - Advanced grid project

### Responsive Design
- **8.2 Responsiveness** - Making layouts responsive
- **8.3 Media Query** - CSS media queries

### Bootstrap Framework
- **11.0 Bootstrap Intro** - Getting started with Bootstrap
- **11.2 Bootstrap Components** - Using Bootstrap components
- **11.3 TinDog Project** - Complete Bootstrap project

### JavaScript
- **16 DOM Challenge** - DOM manipulation exercises
- **17 Dicee Challenge** - Interactive JavaScript game

## Getting Started

Each project folder contains:
- `index.html` - Main HTML file
- `goal.png` - Expected output/design
- `solution.html` - Solution implementation (where applicable)
- Additional assets (CSS, images, etc.)
